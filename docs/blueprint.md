# **App Name**: UltimaMotion60

## Core Features:

- Video Import and Display: Import videos from the device's file system and display them in a customizable 1/2/4 tile grid.
- In-App Video Recording: Record video directly within the app, targeting 720p at 60fps, with feedback on actual achieved settings.
- Slow Motion Playback: Enable slow-motion playback at 1.0, 0.5, 0.25, and 0.125 speeds.
- Basic Transport Controls: Provide play/pause, seek, and speed controls for video playback.
- Tile Selection and Audio Control: Allow users to select an active tile, enabling audio for that tile while muting others.
- Synchronized Playback: Implement a 'Sync' toggle to control play/pause/seek across all tiles simultaneously.
- Local Video Library: Store recorded video clips and metadata locally using IndexedDB for persistent storage.

## Style Guidelines:

- Primary color: A vibrant purple (#9D4EDD) to represent the energy and motion of video, along with the high-end technology involved.
- Background color: A desaturated lavender (#E9D5FF) for a clean, modern backdrop.
- Accent color: A saturated indigo (#5335B5) for buttons, controls, and interactive elements.
- Headline font: 'Space Grotesk', a sans-serif font that provides a modern and techy feel. Body text: 'Inter', a sans-serif font to guarantee good readability.
- Use a set of simple, minimalist icons for video controls (play, pause, seek, etc.) and tile selection.
- Maintain a clean, mobile-first layout with intuitive touch controls and clear visual hierarchy.
- Use subtle transitions and animations for a polished user experience, such as tile selection highlights or play/pause button state changes.
